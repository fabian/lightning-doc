Lightning App

Merklisten mit Push-Benachrichtigung

Ziel des Projekts ist das Erstellen einer iPhone-App zur Verwaltung von Merklisten, welche mit anderen Personen geteilt werden können. Zusätzlich solle eine API mit einem Web-Frontend entwickelt werden. Der Benutzer soll durch ein selbst erarbeitetes Authentifizierungsverfahren ohne Benutzername und Passwort von überall auf seine Merklisten zugreifen können.

Die REST API wird mit dem PHP Framework Symfony entwickelt. Der Source Code wird auf dem zentralen Service GitHub verwaltet und auf der Continuous Integration Platform Travis getestet.
